# Data Descriptions & Summaries
Hier werden die Basics für die Berechnung von statistischen Lagemaßen und Korrelationen zwischen Variablen in Python erklärt.

## Google Colab
[**Repository in Google Colab öffnen**](https://colab.research.google.com/github/flo1304/data_descriptions_summaries/blob/main/workshop.ipynb)

## Lokale Installation
**Anforderungen:**
1. Python Version >= 3.8
2. Editor oder Umgebung bspw.:
    - Jupyter Notebook
    - VS Code + Python Extension
3. Python-Bibliotheken:
    - pandas
    - numpy
    - seaborn
    - matplotlib

**Anleitung:**
1. Repo klonen
    - über Terminal: 
    `https://github.com/flo1304/data_descriptions_summaries`
    und `cd data_descriptions_summaries`
    - oder .zip herunterladen
2. Virtuelle Umgebung
    - `python -m venv venv`
    und `venv\Scripts\activate` (Windows) oder `source venv/bin/activate` (Mac/Linux)
3. Pakete installieren
    - Die benötigten Pakete sollten automatisch beim Starten des Repositories geladen werden
    - falls Fehler auftreten: `pip install pandas numpy seaborn matplotlib jupyter`
4. Notebook starten
